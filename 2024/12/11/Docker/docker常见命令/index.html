docker常见命令

发表于2024-12-11|更新于2024-12-12|Docker
|总字数:416|阅读时长:1分钟|浏览量:

docker常见命令

pull 拉取镜像

images 查看镜像

save 将镜像保存到本地

rmi 删除nginx的镜像

load 安装镜像

创建并运行容器，-d是后台运行

指定输出格式

stop 停止运行容器

start 启动容器

logs 查看容器运行日志

docker log -f nginx（nginx是容器名）不断查看日志

当我访问，日志最后多了几行HTTP请求的日志

docker exec -it nginx bash 进入容器内部用bash操作

查看容器内部的文件，exit 退出容器

进入mysql的容器后，执行 mysql -uroot -p 连接root用户

也可以在容器外直接进入mysql，不用等到进入容器内再 mysql -uroot -p 再进去，前一个mysql是容器名，后面的是进去之后的命令

如果要为命令起别名，去 ~/.bashrc文件夹进行修改，等号左边是执行命令，等号右边是真正执行的命令，保存之后要source一下文件
