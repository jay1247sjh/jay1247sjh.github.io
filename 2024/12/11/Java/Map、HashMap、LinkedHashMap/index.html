Map、HashMap、LinkedHashMap

发表于2024-12-11|更新于2024-12-12|Java
|总字数:792|阅读时长:3分钟|浏览量:

HashMap底层是哈希表，键值对的存储位置只跟键有关，所以跟值是什么类型的没有关系

LinkedHashMap的内部结构较HashMap多了一个双向链表，使得存入和取出的顺序得以一致，另外，跟LinkedHashSet一样，需要重写hashCode方法和equals方法，前者是为了计算哈希值，后者是判断是否添加重复元素
