System工具类

第一个方法 0表示当前虚拟机是正常停止，非0表示当前虚拟机异常停止

第二个方法的返回值是毫秒，我国是东八区，时间原点为1970年1月1日8时

用此方法可以检测程序运行的时间29毫秒（要用long去接收）

相当于把arr1的8个元素拷贝到arr2的1索引之后的位置，实际上是对arr2的覆盖（拷贝长度不能超过原数组本来的长度）

基本数据类型要相同，否则会报错

引用数据类型子类类型可以赋值给基类
