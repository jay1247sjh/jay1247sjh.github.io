环境变量
发表于2024-12-11|Linux
执行命令的时候，会在PATH的这些文件搜索有无指定的命令 通过$查看环境变量的值 当夹杂着其他内容的时候用{}括起来 临时设置变量 vim ~/.bashrc 打开文件在最后一行用export进行编辑命令，并保存 如果想要立即生效就用 source 可以看到此用户生效了（仅仅对当前用户有效） 当我们想要自己去添加一些程序的时候，就自己新建一个文件，并用vim编辑内容 编辑指令内容 之前提到过，想要执行某个文件的内容，必须有x权限，所以这里就先用chmod授权，然后进行指令的使用 vim ~/.bashrc 打开文件添加自定义指令的目录文件夹 自定义命令成功 export PATH=$PATH:~/myenv 是将myenv这个文件夹添加到PATH的路径下，前面那部分的$PATH:不能丢，否则会覆盖掉原有的路径。添加之后，执行命令时就先去PATH文件夹下找相关文件并执行了
